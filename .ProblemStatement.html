Problem Definition
In this given weather app it shows the current status of the weather for specific city
Your goal is to fetch and show the daily forecast of the weather for 8 days.


Notes:
This app uses https://openweathermap.org/ api for fetching the weather details.
The API Key used in this app might not work . you need to create a key(free api subscription) for you
Also you will need to subscribe for daily forecast also.
